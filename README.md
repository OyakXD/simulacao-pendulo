# Simulação de um pêndulo
Projeto desenvolvido para a disciplina de Algebra linear na Universidade Federal do Ceará.

O pêndulo é representado por uma linha conectando os pontos (0, 0) e (0, −5). A cada iteracão, o
pêndulo deve oscilar entre −30◦ e 30◦. Plote o movimento do pêndulo ao longo do tempo.

# Como Usar?

Para usar, vocẽ deve ter o software Octave, abra no GNU e digite 'pendulo'
